## Development

1. Run `bun install` to install dependencies
2. Change `.env.example` to `.env.development` and/or `.env.production` and fill it with your credentials. 
3. Run `bun run deploy:prod` or `bun run deploy:dev` to deploy slash interaction commands.
4. Run `bun run start:prod` or `bun run start:dev` to start your bot
